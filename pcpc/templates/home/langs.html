زبان‌های مجاز برنامه‌نویسی

زبان های برنامه نویسی مجاز برای مسابقه، شامل زبان های C++ ، Python ، Java و C# می باشد. همچنین پارامترهای کامپایل برای هر یک از زبان های برنامه نویسی به صورت زیر خواهد بود.

  • C++ : (GCC 7.5)
  • g++ -XC++ -std=gnu++17-Wall-O2-static-pipe${file}
  • Java: (OpenJDK 1.8)
  • Javac –encoding UTF-8-d. ${file}
  • Pyton3-m Py_compile ${file}
